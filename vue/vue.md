# vue

## vue cli 2.0
```bash
# install
npm install vue
npm install -g @vue/cli-init
```

## vue cli 3.0
```bash
# install
npm install -g @vue/cli-service-global

# create project
vue create my_project

# ui
cd my_project
vue ui

# install dep
vue add @vue/eslint
vue add router
vue add vuex
vue add element

# run dev
npm run serve

# build
npm run build
```
