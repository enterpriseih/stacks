# Git

## Basic

```bash
# install on centos
sudo yum install git

```

## Branch

```bash
# force replace remote branch
git push -f <remote> <branch_name>

# delete remote branch
git push -d origin <branch_name>

# delete local branch
git branch -d origin/bugfix

# rename local branch
git branch -m <new_name>

# replace master with branch
git checkout <branch>
git merge -s ours master
git checkout master
git merge <branch>
```

## Tag

```bash
# use tag to archive
git tag archive/<branch_name> <branch_name>
  
# delete tag
git tag -d oldtag

# rename tag
git tag <new_tag> <old_tag>
git tag -d <old_tag>
```

## Remote

```bash
# show remote url
git remote -v

# set remote url
git remote set-url <repo>
git remote set-url --add origin <repo>
git remote set-url --delete origin <repo>

# add to github
git remote add github <repo>
git push github master
git branch -u github/master master
git remote set-url origin <repo>
```

## Submodule

```bash
# add other project as submodule
git submodule add <repo> <local_dir>

# init submodule
git submodule init

# update submodule
git submodule update

# delete submodule
git submodule rm <local_dir>
```
